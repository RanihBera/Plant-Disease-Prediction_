# 🌿 Plant Disease Prediction System

## Overview
This is a deep learning-based web application that predicts plant leaf diseases using a Convolutional Neural Network (CNN). The application is built with Streamlit and trained on the PlantVillage dataset.

## Features
- Upload plant leaf images
- Predict plant diseases
- Display confidence score
- Show disease symptoms
- Show disease causes
- Show prevention methods
- Show treatment recommendations

## Supported Plants
- 🍅 Tomato
- 🥔 Potato
- 🌶 Pepper

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow

## Dataset
PlantVillage Dataset

## Model
CNN (Convolutional Neural Network)

## How to Run

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
